## ▶️ How to Run a Project

1. Clone the repository

```bash
git clone https://github.com/Shreyas-coding-master001/My-Animation-Journey.git
```

2. Open the project

```bash
cd My-Animation-Journey
```

3. Navigate to any day's folder

Example:

```bash
cd Day1
```

4. Install dependencies (first time only)

```bash
npm install
```

5. Start the development server

```bash
npm run dev
```

6. Can Check All the routes for all Animation as per Daily Progress
